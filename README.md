# 📊 Sales Performance Dashboard (Excel)
Excel-based Sales Dashboard using Superstore Dataset
## 📌 Overview
This project analyzes retail sales data using Excel and presents insights through an interactive dashboard.

## 📂 Dataset
Superstore Dataset (Kaggle)  
https://www.kaggle.com/datasets/vivek468/superstore-dataset-final

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers

## 📊 Features
- Dynamic KPI cards (Sales, Profit, Orders,profit margin%)
- Interactive filters using slicers
- Sales trend analysis over time
- Region and category-wise performance

## 💡 Insights
- West region generated highest sales  
- Technology category most profitable  
- Phones and Chairs are top-selling products
